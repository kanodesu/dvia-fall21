What Attracts You More?

A data visualization project based on the female user profiles of a dating website Lovoo.
It includes user profile data such as the number of profile visits, user age, etc.

VIEW









The dataset contains only user profile infos of female users on the website Lovoo. It includes user profile data such as the number of profile visits, user age, location, etc. When Dating apps like Tinder were becoming viral, people wanted to have the best profile in order to get more matches and more potential encounters.

But I think like all data online, especially in people’s profiles could be ‘fake’ in order to gain more attention, and I think this can be connected to the topic, illusion.

I am wondering at what makes a profile on a dating app popular, and am going to compare the counts of profile visits and likes with some other factors, such as gender, age, and pictures the user put in her/his profile.
